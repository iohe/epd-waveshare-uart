# Examples:

All of these examples are projects of their own. 

A few notes:
 - If not stated otherwise the example is for a Raspberry Pi running Linux.
    

### epd4in3

Connect epd4in3 display to Raspberry Pi 4:
- BUSY -> A10
- RST -> A9
- DC -> A8
- CS -> B12
- CLK -> B13
- DIN -> B15
- GND -> G
- VCC -> 3.3
